# How to create a template repository
Purpose: steps for creating a GitHub Template Repo

* **Create a repo:** Create a GitHub Repository or use an existing repository you have. <br />
    _If you need help, check out the [GitHub documentation on repo creation](https://docs.github.com/en/get-started/quickstart/create-a-repo)_

* **Convert repo to a template:**
  * Under your repository name, click 'Settings' tab
  * Select 'Template repository' checkbox
  * To confirm it worked, select 'Code' tab and you'll see that the repository is now a template
